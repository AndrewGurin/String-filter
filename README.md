# **STRING FILTER**
### *By Andrew Gurin*
v.1.0 (2022-08-04)
## Описание
Программа "String filter" предназначена для фильтрации введённых пользователем строк по их длине. В данной версии используется статическое пороговое значение фильтрации — не более 3 символов. В следующих версиях планируется добавить возможность пользовательского ввода порогового значения. Ожидайте обновления!
## *Features*
* Возможность ввода до 10 строк включительно;
* За счёт ограниченного количества вводимых строк нет необходимости пользователю его предварительно указывать. Достаточно по окончании ввода оставить поле пустым и нажать клавишу Enter;
* Программа показывает как исходный массив строк, так и отфильтрованный, благодаря чему пользователь может проверить корректность введённых им строк.
## Instructions
1. Клонируйте репозиторий с программой с сайта GitHub;
2. Перейдите в терминале в папку с программой;
3. Запустите программу посредством команды "dotnet run";
4. По окончании ввода каждой строки нажимайте Enter. Программа промежуточно выводит количество оставшихся свободных ячеек массива;
5. При полном заполнении ячеек массива либо при пустом вводе программа приступает к фильтрации и выводит новый отфильтрованный массив строк.
